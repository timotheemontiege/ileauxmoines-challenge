# Fichiers GPX d'exemple

## `tour-exemple.gpx`

Trace synthétique d'**un tour complet** de l'Île-aux-Moines, conçue pour tester
l'upload et l'algorithme de détection immédiatement.

- 1441 points (~1 point/seconde)
- 40 premiers points = court trajet « voiture » **hors** Golfe → filtrés
  automatiquement par la bounding box (démonstration du filtrage du bruit)
- Tour attendu : **≈ 00:22:46**, **≈ 9.3 km**, **≈ 13.2 nœuds**

Utilisation : page **Soumettre** → choisir ce fichier → catégorie au choix.
Résultat attendu : `Tour détecté ✓`.
